## create a game screen

h1 rulette

p A rule stacking party game based on S7E7 and S8E2 of Game Changer on DropoutTV. Playable live or over voice chat with 3-6+ players.

p Sign up for Home Edition on Kickstarter.

h2 rules

Players start with 20 points.
Spin on your turn to acquire rules.
Click to accuse another player of breaking theirs.
Host decides how many points to deduct (if any).

h2 getting started

Enter a name to create a game as host.
Scan QR code or click shared link to join as player.
Host may start the game once two or more non-host players have joined.
