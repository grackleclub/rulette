rulette

A rule stacking game based on S7E7 and S8E2 of Game Changer on Dropout.

Sign up for Home Edition on Kickstarter.

rules

  • Players start with 20 points.
  • Spin on your turn to acquire rules for yourself.
  • Accuse other players of breaking their rules.
  • Host decides point penalties.

getting started

  • Enter a game name to begin hosting a new game.
  • Scan QR code or click shared link to join as player.
  • Host may start the game once two or more non-host players have joined.
